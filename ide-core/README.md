# IDE Core - EL CUERPO

Aplicación visual estilo Visual Studio para Portul.

## Módulos

- **editor/**: Motor de texto, renderizado, syntax highlighting
- **ui/**: Paneles, explorador de archivos, terminal integrada
- **lsp/**: Language Server Protocol - puente entre editor y compilador
- **workspace/**: Gestión de proyectos, archivos y configuración `.portulrc`
- **assets/**: Iconos, temas (dark mode por defecto)

## Características

✅ Syntax highlighting nativo para Portul
✅ Integración con Language Server Protocol
✅ Terminal integrada
✅ Explorador de archivos visual
✅ Tema dark mode por defecto
✅ Configuración por proyecto (`.portulrc`)

## Stack Tecnológico

- Frontend: Electron + React (o similar)
- Backend: Node.js + LSP
- Estilos: Tailwind CSS o similar

## Desarrollo

```bash
npm install
npm start
```
